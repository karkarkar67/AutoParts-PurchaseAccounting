# АИС «АвтоЗапчасти: Учёт закупок»

Система для автоматизации учёта закупок запасных частей у поставщиков.

## Функциональность
- Ведение справочника поставщиков (название, адрес, телефон)
- Ведение каталога деталей (наименование, артикул, цена)
- Регистрация фактов покупки (дата, количество)
- Автоматический расчёт суммы закупки
- Сравнение цен на одинаковые детали от разных поставщиков
- Формирование отчётов по закупкам и поставщикам

## Технологии
- Python / Django (или C#/.NET)
- PostgreSQL
- Git
